# CIT4224: Internet and Web Publishing

## Overview
This repository contains a comprehensive collection of web development labs, class exercises, and project work. It covers the full stack of web publishing, from basic HTML structure to dynamic server-side scripting with PHP.

## Key Technologies
- **HTML5**: Semantic structure and page layout.
- **CSS3**: Responsive design, flexbox, and advanced styling.
- **JavaScript & jQuery**: Client-side interactivity and DOM manipulation.
- **PHP**: Server-side logic and form handling.

## Directory Structure
- `Labs/`: Weekly assignments covering specific web technologies (HTML, CSS, JS, PHP).
- `project/`: A complete web application project, including user dashboards and database interactions.
- `Class code/`: Snippets and examples from lecture demonstrations.
- `Lab 9/`: Advanced exercises focused on advanced CSS and layout techniques.

## How to View
Most files can be viewed directly in a web browser. PHP files require a local server environment like XAMPP or MAMP.
